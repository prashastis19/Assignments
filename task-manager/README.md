# Task Manager (Node.js Assignment)

A simple command-line Task Manager built using Node.js  

# Project Structure

The repository is structured as follows:

task-manager:

1.app.js

2.tasks.json

3.README.md
##  Features

The application supports the following operations:

Add a Task: Creates a new task with a unique ID and sets its status to pending.

List Tasks: Displays all tasks, showing their ID, status, and label.

Remove a Task: Deletes a task based on its unique ID.

Update a Task: Modifies the label of an existing task by its ID.

Mark as Done: Changes the status of a task to done.
